How It Works

Every Meshtastic node is a virtual IRC user.

• Connect: mesh-dash:6667

• Your nick is automatically set to your node's LongName-ShortName

• BNC: Messages are buffered when offline. Auto-replayed on join!

• IRC Commands: /REPLAYBUFFER - replay all buffered | /CLEARBUFFER - clear buffer

• Use /WHOIS nodename to see node info (last heard, hardware)
